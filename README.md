# Powershell-Utilities
Basic utilities written in powershell.

### Compare-MD5Sum.ps1
> Similar to the linux `md5sum` utility, this script gets the MD5 hash of a file and/or compares the hash with the *.asc/.md5 file.

### Get-SIDFromUser.ps1
